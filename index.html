Second Order Elementary Cellular Automata

Cellular automata are very interesting things. These are elementary cellular automata, starting from a random seed. You can try out different automata rules by typing a new number in the box and hitting Enter (some favorites to try are 18, 30, and 184). If you select the "Second order" box you will see that same rule rendered in second order. For more information on what that means, check out this wikipedia article: https://en.wikipedia.org/wiki/Second-order_cellular_automaton
